# jquery.deserialize
Native jQuery deserializer
## Install
```javascript
<script src="jquery.min.js"></script>
<script src="../jquery.deserialize.min.js"></script>
```
or 
```javascript
bower install deserialize
```
## Usage
your data can be string or object
```javascript
$("form").deserialize(data);
```
